## Description

This "Secure Notes" App demonstrates CI/CD and secret handling by having a Frontend where a note is typed. The note will then be encrypted and the encrypted note will be displayed. The encryption key is handled securely using Azure KeyVault.
Additionally, GitHub Actions is used for CI/CD.

## Goal

The goal of this project is to demonstrate CI/CD and secret handling for educational purposes, showcased with a "Secure Notes" App.
